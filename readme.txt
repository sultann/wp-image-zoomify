WP Image Zoomify is simple image light box plugin with zoom effect.


== Description ==

The plugin adds simple light box function with zoom effect to any of the image on your site.

Usages:
Just add `rel='zoomify'` or add a class `zoomify` to the image.
Example:
Using rel attribute
`<img rel="zoomify" src="http://yoursite/image.jpg" class="alignnone size-medium wp-image-1263" />`
Using class
`<img src="http://yoursite/image.jpg" class="zoomify" />`

The plugin will automatically recognise the image and add light box to that.


== Installation ==

Use the automatic installer on the 'Plugins' page, or upload the unzipped plugin directory to your `/wp-content/plugins/` directory and activate it.
